CMDragons 02 Tactics Help

Introduction

There are a whole range of low-level tactics for running the robots. The GUI can call these tactics directly to aid in testing the low-level routines. To execute a tactic, select on a robot, then go to teh tactic box at the top of the GUI. Click on the drop down list and select the tactic you want. Add any parameters to the right hand edit box if you need them. Then click on Go. Likewise, click on Stop to stop the tactic from executing.

Tactics

The tactic parameters and operation are:

  • None: do nothing
  • NULL: execute the default tactic...soon to be depricated
  • stop: do the halt and stand still tactic
  • shoot: execute a shoot: parameters ..????
  • pass: pass to another robot
  • goalie: execute the goalie behavior
  • block
  • defend cross
  • latency - performs the latency testing
  • test_step < vx vy va filename > test the step response of a robot records the results to filename
  • test_ramp < vx vy va ax ay aa filename > tests the ramp response of a robot. It ramps with constant acceleration a to peak velocity v and stores the resultsl to the file filename
